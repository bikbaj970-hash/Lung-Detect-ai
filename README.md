
Lung Detect AI

AI-powered lung disease detection from medical images.

🔗 Live Demo: https://replit.com/join/inrzodlpjemfluf-dhilangurtoo

What It Does

Lung Detect AI analyzes lung images using artificial intelligence to identify potential cases of tuberculosis, pneumonia, COVID-19, or healthy lungs. The application helps provide fast image-based predictions, making it useful for learning, research, and early screening demonstrations.
